# 即梦风格 UI 改造规划

## 背景分析

根据提供的即梦AI界面截图，需要将现有系统改造为以下风格：

### 即梦核心设计特征
1. **Agent 模式输入框**：大标题 + 简洁输入区 + 底部参数按钮 + 右侧操作按钮
2. **创作类型菜单**：下拉式选择器，支持多种创作类型切换
3. **时间线历史记录**：按日期分组的瀑布流网格展示
4. **作品详情页**：左右分栏布局，左侧大图/播放器 + 右侧参数信息面板

## 改造目标

### 1. 图片生成工作台改造
**当前状态**：传统表单布局，多区域分散
**目标状态**：即梦 Agent 模式风格

**具体改造**：
- 顶部显示大标题"开启你的 Agent 模式 · 即刻造梦！"
- 添加创作类型下拉菜单（Agent 模式、图片生成、视频生成、数字人、配音生成、动作模仿）
- 单行大输入框替代现有的分散表单
- 输入框左侧添加附件上传按钮（图片上传）
- 输入框底部添加参数按钮（点击展开参数面板）
- 输入框右侧添加语音输入和提交按钮
- 参数通过折叠面板展示：分辨率、宽高比、Seed、Scale 等
- 生成结果以卡片网格展示，替代传统列表
- 移除"帧类型"、"关联镜头"等复杂选项，整合到参数面板

### 2. 视频生成工作台改造
**当前状态**：传统表单布局
**目标状态**：即梦 Agent 模式风格

**具体改造**：
- 顶部显示大标题"开启你的 Agent 模式 · 即刻造梦！"
- 添加创作类型下拉菜单
- 单行大输入框替代现有的分散表单
- 输入框左侧添加附件上传按钮（首帧/尾帧图片上传）
- 输入框底部添加参数按钮（点击展开参数面板）
- 输入框右侧添加语音输入和提交按钮
- 参数通过折叠面板展示：时长（5秒/10秒）、宽高比、Seed 等
- 活跃任务以卡片网格展示

### 3. 作品历史页面改造
**当前状态**：表格列表视图
**目标状态**：即梦时间线瀑布流

**具体改造**：
- 图片 Tab 和视频 Tab 保持不变
- 内容改为按日期分组的瀑布流网格（如"3月14日"、"1月2日"）
- 每个作品以缩略图展示，正方形/矩形自适应
- Hover 时显示操作按钮（预览、下载、删除）
- 支持批量操作功能
- 移除传统的表格列表视图

### 4. 图片详情页改造
**当前状态**：任务信息表格形式
**目标状态**：即梦作品展示风格

**具体改造**：
- 左侧：大图展示区，支持缩放、全屏查看
- 右侧：作品信息面板
  - 顶部操作栏：下载按钮、收藏星标、分享、更多操作（...）
  - 图片提示词区域（展示完整 prompt）
  - 模型信息行：智链参考 | 图片 4.1 | 1:1 | 2K | 详细信息
  - 操作按钮组（2列网格布局）：
    - 生成视频
    - 去画布编辑
    - 编辑超清
    - 细节修复
    - 扩图
    - 消除笔
    - 对口型
    - 图片编辑器
    - 重新编辑
    - 再次生成
  - 生成历史记录（折叠面板）

### 5. 视频详情页改造
**当前状态**：任务信息卡片形式
**目标状态**：即梦作品展示风格

**具体改造**：
- 左侧：视频播放器，支持进度条、音量、全屏、倍速
- 右侧：作品信息面板
  - 顶部操作栏：下载按钮、收藏星标、分享、更多操作（...）
  - 视频提示词区域（展示完整 prompt）
  - 模型信息行：智链美颜 | Seedsance 1.5 Pro | 12s | 详细信息
  - 操作按钮组（2列网格布局）：
    - 对口型
    - AI 音效
    - AI 配乐
    - 补帧
    - 智能超清
    - 重新编辑
    - 再次生成
  - 生成历史记录（折叠面板）

## 技术实现方案

### 新增组件
1. **JimengInput.tsx**：即梦风格 Agent 输入框组件
2. **CreationTypeMenu.tsx**：创作类型下拉菜单组件
3. **ParamPanel.tsx**：参数折叠面板组件
4. **TimelineGrid.tsx**：时间线瀑布流网格组件
5. **ImageDetailPanel.tsx**：图片详情右侧面板组件
6. **VideoDetailPanel.tsx**：视频详情右侧面板组件

### 修改页面
1. **ImageGeneration.tsx**：完全重构为即梦风格
2. **VideoGeneration.tsx**：完全重构为即梦风格
3. **GenerationHistory.tsx**：重构为时间线瀑布流
4. **TaskDetail.tsx**：重构为即梦详情页（或拆分为 ImageDetail.tsx 和 VideoDetail.tsx）

### 路由调整
- `/content/image-detail/:id`：图片详情页
- `/content/video-detail/:id`：视频详情页
- 保持现有 `/content/task/:id` 路由兼容

## 实施优先级

1. **第一阶段**：创建共享 UI 组件（JimengInput、CreationTypeMenu、ParamPanel、TimelineGrid）
2. **第二阶段**：改造图片生成工作台和视频生成工作台
3. **第三阶段**：改造历史页面为时间线瀑布流
4. **第四阶段**：创建图片详情页和视频详情页
5. **第五阶段**：更新路由和菜单，验证测试

## 注意事项

1. 保持与现有 Store 的兼容性
2. 确保 TypeScript 类型安全
3. 保留所有现有功能，只是 UI 改造
4. 深色模式适配
5. 响应式布局支持
