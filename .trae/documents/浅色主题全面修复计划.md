# 浅色主题全面修复计划

## 问题诊断

### 根本原因
**17个页面文件 + 3个组件文件** 全部使用**硬编码深色类名**，没有 `dark:` 前缀。在浅色模式下，这些类名直接生效导致：
- 文字颜色 `text-gray-100/200/300` 在白色背景上几乎不可见
- 边框 `border-gray-800` 在浅色背景上过深
- 背景 `bg-gray-700/800` 在浅色模式下显示为深灰块
- hover 效果 `hover:bg-gray-700` 在浅色模式变成深色

## 修复规则（全局替换映射表）

### 颜色类名替换规则

| 硬编码深色类 | 替换为（双主题） |
|-------------|-----------------|
| `text-gray-100` | `text-gray-900 dark:text-gray-100` |
| `text-gray-200` | `text-gray-800 dark:text-gray-200` |
| `text-gray-300` | `text-gray-600 dark:text-gray-300` |
| `text-gray-400`（图标/次要文字） | `text-gray-500 dark:text-gray-400` |
| `border-gray-800` | `border-gray-200 dark:border-gray-800` |
| `border-gray-800/50` | `border-gray-200 dark:border-gray-800/50` |
| `border-gray-700` | `border-gray-300 dark:border-gray-700` |
| `border-gray-600` | `border-gray-300 dark:border-gray-600` |
| `bg-gray-800` | `bg-gray-100 dark:bg-gray-800` |
| `bg-gray-800/50` | `bg-gray-100/50 dark:bg-gray-800/50` |
| `bg-gray-900/50` | `bg-gray-100 dark:bg-gray-900/50` |
| `bg-gray-700` | `bg-gray-200 dark:bg-gray-700` |
| `hover:bg-gray-800/30` | `hover:bg-gray-100 dark:hover:bg-gray-800/30` |
| `hover:bg-gray-700` | `hover:bg-gray-100 dark:hover:bg-gray-700` |
| `hover:bg-gray-700/50` | `hover:bg-gray-100 dark:hover:bg-gray-700/50` |
| `hover:bg-gray-600` | `hover:bg-gray-200 dark:hover:bg-gray-600` |
| `divide-gray-800` | `divide-gray-200 dark:divide-gray-800` |

### 特殊组件替换

**宽高比/模式选择按钮：**
```
'bg-gray-700 hover:bg-gray-600 text-gray-300'
→ 'bg-gray-200 dark:bg-gray-700 hover:bg-gray-300 dark:hover:bg-gray-600 text-gray-700 dark:text-gray-300'

'bg-gray-800 text-gray-400 hover:bg-gray-700'
→ 'bg-gray-100 dark:bg-gray-800 text-gray-600 dark:text-gray-400 hover:bg-gray-200 dark:hover:bg-gray-700'
```

**Pagination 组件：**
```
'bg-gray-800 hover:bg-gray-700 text-gray-300'
→ 'bg-gray-100 dark:bg-gray-800 hover:bg-gray-200 dark:hover:bg-gray-700 text-gray-700 dark:text-gray-300'

'bg-gray-800 border border-gray-700 rounded text-sm'
→ 'bg-white dark:bg-gray-800 border border-gray-300 dark:border-gray-700 rounded text-sm text-gray-900 dark:text-gray-100'
```

**Progress 进度条：**
```
'bg-gray-700' (轨道)
→ 'bg-gray-200 dark:bg-gray-700'
```

**Roles 权限标签：**
```
'bg-gray-800 rounded text-gray-300'
→ 'bg-gray-100 dark:bg-gray-800 rounded text-gray-700 dark:text-gray-300'
```
