# 优化菜单名称 Spec

## Why
当前内容创作菜单下的名称（图片生成、视频生成、图片历史、生成历史、资产管理、镜头管理、首图/尾图）过于直白和土气，需要更具创意和专业感的命名，提升系统的整体质感。

## What Changes
- 重命名侧边栏菜单项，使用更具创意和专业感的名称
- 保持路径不变，仅修改显示名称

## Impact
- Affected specs: 导航系统
- Affected code: `src/components/Sidebar.tsx`

## MODIFIED Requirements

### Requirement: 内容创作菜单名称优化

**当前名称 → 新名称**：

| 当前名称 | 建议新名称 | 说明 |
|----------|------------|------|
| 图片生成 | **灵感创作** | 体现创意和创作过程 |
| 视频生成 | **动态创作** | 体现动态/视频生成的特性 |
| 图片历史 | **灵感画廊** | 更具艺术感的名称 |
| 生成历史 | **时光回溯** | 体现历史回顾的概念 |
| 资产管理 | **素材仓库** | 更形象地表达资产存储 |
| 镜头管理 | **分镜脚本** | 更专业的影视术语 |
| 首图/尾图 | **关键帧** | 更专业的技术术语 |

#### Scenario: 用户查看菜单
- **WHEN** 用户展开"内容创作"菜单
- **THEN** 看到更具专业感和创意的菜单名称
