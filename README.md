# AIGC Asset Management Demo

一个面向内容生产流程的 AIGC 数字资产管理前端演示项目，覆盖内容中心、项目中心、系统管理三大模块，并提供镜头级创作工作台、项目镜头编排、图片/视频生成结果追溯等能力。

## 功能概览

- 内容中心：图片创作、视频创作、镜头管理、首图/尾图管理、资产管理
- 项目中心：客户、品牌、项目、提案、任务、审核与项目详情工作台
- 系统管理：成员、角色权限、系统日志、系统设置
- 镜头工作台：在镜头上下文中直接完成图片/视频创作，并选择首图、尾图、最终视频

## 本地运行

```bash
npm install
npm run dev
```

默认开发地址：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/` 目录。

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- Radix UI

## 说明

- 当前仓库主要用于前端演示与交互验证。
- 图片、视频生成链路为前端模拟流程，便于展示镜头级创作与结果管理体验。
